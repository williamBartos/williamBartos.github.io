Raspberry Pi Weather Logging

Description

• Language, Notable Libraries:

>_Python 3.5
>_Plotly
>_Adafruit_Python_DHT
• Hardware:

>_Raspberry Pi 3 Model B

>_DHT11 Temperature and Humidity Sensor

>_DS18B20 Waterproof Temperature Probe


• Motivation and Objectives:

As an engineer, I love working with my hands. I had an itch to create a useful project that integrated hardware and software. My engineering Capstone project involved using an RPi to create an automated Sequencing Bioreactor for wastewater treatment which t was a great foray into controls and hardware. I wanted to start another fun weekend project that is hands on.

My immediate objectives of the project was to design a sensor suite to measure outdoor temperature, indoor temperature and humidity. The first part of the process was wiring the sensors to the Raspberry Pi GPIO pins. A simple circuit in a pull up congfiration was utilized for each sensor. The software controlling and logging the data was written in Python 3.5. The waterproof sensor has an easy output and can be accessed directly without any munging or API's. The DHT11 sensor is more complex in how it transmits data to the computer. Thankfully, Adafruit has an API on GitHub that made extracting indoor temperature and humidity data easy.

To plot the data online in real time, I used Plotly. Plotly allowed me to get the data online painlessly and quickly in a well formatted interface. Plotly also offers built in data analysis tools and the ability to embed the graph into my website.

• Future Objectives:

Currently I am not storing the data I'm collecting. The next step in this project is to create a database to store and retrieve the sensor data.

My long term future goal is to take what I've learned in this project and apply it to automating my garden. I want to be able to automatically control parameters like irrigation and also utilize data to assess the condition of my crops and make informed growing decisions. To this end I've purchased an 8 switch relay and have been able to successfully control 120V appliances through my Rpi.
